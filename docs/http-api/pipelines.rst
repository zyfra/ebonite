Update pipeline
---------------

Request
^^^^^^^

.. code-block::

  PATCH /pipelines/<:id>

* `id`: id of pipeline to update

.. code-block:: json

  {
    "name": "first pipeline"
  }

Response
^^^^^^^^^^^^^^

* `204`: OK
* `404`: if given pipeline doesn't exist

Delete pipeline
---------------

Request
^^^^^^^

.. code-block::

  DELETE /pipelines/<:id>

* `id`: id of pipeline to delete

Response
^^^^^^^^^^^^^^

* `204`: OK
* `404`: if given pipeline doesn't exist
